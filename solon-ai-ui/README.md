### solon ai ui 模块描述

本模块，主要为了解决后端（Agent，LLM程序）和前端对接协议（如何展示、SSE以什么样格式传输？）的问题，通过对接业界主流UI协议或者提供快速对接的框架，使得业务的前端能够尽快落地。

### solon ai ui 内置清单


| 依赖包               | 适配协议   | 描述                                                              |
|-------------------|--------|-----------------------------------------------------------------|
| solon-ai-ui-aisdk | AI SDK | 提供基于Vercel AI SDK的标准化UI协议对接，支持AI-Elements组件库和AI-Elements-Vue组件库 |
| solon-ai-ui-agui  | AG-UI  | 提供基于AG-UI的标准化UI协议对接，便于快速的对接组件库/复用成熟设计                           |
